# 🚀 Smart Parking V3 - Final Submission

Welcome to the **Smart Parking System**, a state-of-the-art, architecture-grade parking management platform. This project was developed as a final submission for the 4th-semester Software Engineering course (FA24).

---

## 💎 Project Essence
This application delivers a high-end, responsive, and data-driven experience for modern parking ecosystems. It features full **Dynamic Pricing**, **Digital Wallet** integration, **Loyalty Reward** algorithms, and **Automated Overstay Fine** logic—all presented through a visually stunning "Vibrant V3" aesthetic.

## 🛠️ Technology Stack
*   **Frontend**: React.js 18 + Vite (Ultralight & High Speed)
*   **Styling**: Tailwind CSS v4 (Modern Utility-First Design)
*   **Animations**: Framer Motion (Physics-based UI transitions)
*   **Visualizations**: Chart.js (Real-time admin analytics)
*   **Iconography**: Lucide React
*   **Typography**: Google Fonts (Outfit & Inter)

---

## 🌟 Key Features
1.  **Hyper-Modern Auth**: Glassmorphic registration and login with referral code integration.
2.  **Visual Matrix Booking**: Real-time "Driveway Grid" (PN001 style) with Red/Green occupancy indicators.
3.  **Dynamic Surge Pricing**: Automatic 20% price increase when a zone exceeds 80% occupancy.
4.  **Loyalty Engine**: 
    *   10 pts per 1 PKR spent.
    *   +200 pts per referral (both parties).
    *   +10 pts "Evening Bonus" for bookings after 18:00.
5.  **Digital Wallet**: Secure balance management and internal transaction logging.
6.  **Admin Command Center**: Neon status blocks and weekly revenue analytics charts.

## 🚀 Getting Started

### Prerequisites
*   [Node.js](https://nodejs.org/) (Version 16 or higher)
*   [npm](https://www.npmjs.com/) (Standard Node Package Manager)

### Installation
1.  **Clone/Extract** the project folder.
2.  Open your terminal inside the project directory:
    ```bash
    npm install
    ```
3.  **Launch the System**:
    ```bash
    npm run dev
    ```
4.  Access the interface at: `http://localhost:5173`
